Not long ago, finding a place to be an entrepreneur at Rutgers was hard. There was no network, no advice, and certainly no money. As entrepreneurs, we wanted to fix that. So we started RESx, a program run by the Rutgers Entrepreneurial Society dedicated to helping Rutgers startups succeed. Want to join?

RESx is a 10-week accelerator program which runs 3 times per year. Our next session is taking place January - April and will be run by Nick and Aidan

The accelerator helps young startups of any type (tech, social, hardware, etc.) develop a great product, connect with mentors, raise capital, and launch to the world. We are centered at Rutgers, but are not focused there.
